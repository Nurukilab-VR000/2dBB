# 詳細
パドル：ボールを跳ね返す役割</Br>
ボール：ブロック、パドルに当たると跳ね返る</Br>
ブロック：種類、色、配置、耐久力がある</Br>
アイテム：ブロックを壊すと一定確率でアイテムが落ちる</Br>

①：パドルの長さを1.5倍にする</Br>
②：ボールの大きさを2.0倍にする</Br>
③：ボールの個数を2個にする</Br>

※GameManager.csで全ての状態を管理出来るようにしました。
